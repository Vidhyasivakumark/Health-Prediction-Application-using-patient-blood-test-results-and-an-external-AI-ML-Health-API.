# AI-Based Health Prediction Application

## Overview

This project is an AI/ML-powered healthcare prediction application developed using Python and Streamlit. The system predicts patient health risk based on blood test parameters such as glucose, haemoglobin, and cholesterol levels.

The application also supports CRUD operations for patient record management and integrates APIs for prediction and healthcare processing.

---

## Features

* Patient data entry form
* AI/ML-based disease prediction
* Real-time prediction results
* CRUD operations (Create, Read, Update, Delete)
* Streamlit interactive UI
* API integration
* Input validation
* Patient record storage and management

---

## Technologies Used

* Python
* Streamlit
* Machine Learning
* Scikit-learn
* Pandas
* NumPy
* REST API
* SQLite / Database

---

## Project Workflow

1. Enter patient details
2. Validate user input
3. Predict disease/risk using ML model
4. Display prediction result
5. Save and manage patient records

## Input Parameters

* Cholesterol Level
